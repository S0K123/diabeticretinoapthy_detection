# Diabetic Retinopathy Detection

## Project Description
Diabetic retinopathy is a diabetes complication that affects the eyes. This project aims to develop a system that accurately detects diabetic retinopathy in images of the retina using machine learning techniques.

## Tech Stack
- **Programming Language:** Python
- **Frameworks:** TensorFlow, Keras
- **Libraries:** OpenCV, scikit-learn, NumPy, Matplotlib
- **Database:** SQLite

## Key Contributions
- Developed a robust machine learning model to identify diabetic retinopathy.
- Created an intuitive user interface to simplify the interaction with the system.
- Conducted extensive testing to ensure accuracy and reliability of the detection system.

## Outcomes/Impact
- The system contributes significantly to early detection and treatment of diabetic retinopathy, potentially preventing vision loss in patients.
- Increased awareness around diabetic eye diseases among healthcare professionals and patients.

## Instructions to Run the System
1. **Clone the Repository:**  
   ```bash
   git clone https://github.com/S0K123/diabeticretinoapthy_detection.git
   cd diabeticretinoapthy_detection
   ```  
2. **Install Dependencies:**  
   ```bash
   pip install -r requirements.txt
   ```  
3. **Run the Application:**  
   ```bash
   python app.py
   ```  
4. **Access the system through your web browser at:**  
   `http://localhost:5000`
